# Boss Rush Niche Genre Opportunity

2026년 기준 Steam 저경쟁 실험 니치. "보스전만" 집중 구조로 소규모 개발에 적합.

---

## 장르 정의

- **Boss Rush**: 일반 적 없음, 연속 보스전만으로 구성된 게임
- 핵심 경험: 패턴 학습 → 실행 → 숙달의 루프
- 차별점: 일반 액션/roguelite 대비 "빌드업" 없이 즉시 하이라이트만
- 대표작: Furi (2016) · Cuphead (보스전 비중) · Hades (보스전 구조)

---

## 2026 시장 현황

**저경쟁 확인** (confidence: medium):
- 2026 트렌드 분석에서 "실험적 니치"로 명시 언급
- "highly focused boss-rush structures" = 공급 부족 구간
- itch.io에서 보스러시 태그 게임 수 적음

**활성 타이틀 예시 (2025-2026)**:
- VOIDFACE — 불릿헬 보스러시 슈터
- NanoApostle — 다크 SF 보스러시 (Destruction Point 시스템)
- Perennial Order — 2D 식물 공포 보스러시
- IDLE BOSS RUSH — 아이들 장르 퓨전 (장르 실험)

> [!gap] Boss Rush 장르의 연간 출시 정확 수치는 SteamDB 직접 조회 필요. 저경쟁은 현재 "medium confidence".

## 개발 기간 추정

| 방식 | 기간 | 이유 |
|---|---|---|
| 솔로 (단순 아트) | **4-8개월** | 보스 디자인이 전부 = 패턴 설계 집중 |
| 솔로 (고완성도) | **8-14개월** | Furi급 비주얼/음악 |
| 소규모 팀 | **6-12개월** | |

**단기 개발 유리한 이유**:
- 월드 빌딩, 탐험 맵, 잡몹 AI 불필요
- 콘텐츠 = 보스 수 × 패턴 다양성 → 범위 명확
- 각 보스 = 독립 레벨 → 점진적 출시(EA) 용이

## 장르 퓨전 기회

보스러시는 단독으로도 성립하지만 다른 장르와 퓨전 시 차별화 용이:

| 퓨전 | 사례 | 효과 |
|---|---|---|
| 보스러시 + Roguelite | Hades 구조 | 매 런 다른 순서·변형 |
| 보스러시 + 불릿헬 | VOIDFACE | 탄막 + 패턴 이중 도전 |
| 보스러시 + 아이들 | IDLE BOSS RUSH | 경쟁 낮은 퓨전 실험 |
| 보스러시 + 내러티브 | Furi | 보스 = 스토리 챕터 |

---

## 성공 조건

1. **보스 패턴의 독창성** — 장르 전체가 보스에 집중 → 패턴 품질이 평가의 전부
2. **진입 신호 (텔레그래프)** — 공정한 어렵움 = 패턴 학습 가능성
3. **유니크한 아트 또는 테마** — 동일 메카닉 → 비주얼로 차별화
4. **적절한 분량** — 보스 6-10개 = 적절한 가격대 ($10-15)

## Echo 관련성

- **높음** — Echo(런앤건)의 보스 디자인에 직접 적용 가능
