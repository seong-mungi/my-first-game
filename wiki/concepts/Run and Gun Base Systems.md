# Run and Gun Base Systems

## 장르의 최소 기능 세트 (Minimum Viable Run and Gun)

런앤건 장르의 **핵심 메커닉(load-bearing core)**은 다음 5가지다.
이 5가지가 있으면 런앤건이고, 없으면 다른 장르다.

```
┌─────────────────────────────────────────────────┐
│          런앤건 최소 기능 세트 (MVS)              │
│                                                 │
│  1. 횡스크롤 2D 이동 (Move)                      │
│  2. 점프 (Jump)                                 │
│  3. 사격 — 이동 중 동시 가능 (Shoot-while-moving) │
│  4. 웨이브/밀집 적 패턴 (Enemy Waves)             │
│  5. 피격 = 즉각적 대가 (Hit = Consequence)       │
└─────────────────────────────────────────────────┘
```

## 기반 시스템 상세 분석

### 1. 횡스크롤 이동 (Movement)

**핵심 특성:**
- 좌우 이동 + 점프 = 2D 공간 내 전체 신체 제어.
- 이동 자체가 생존 전략 — "달리기(Run)"가 장르명의 절반.
- 이동 속도는 장르의 템포를 결정. Metal Slug: 중간속도(탐험 허용). Contra: 빠른 속도(전진 강조).

**필수 이동 서브시스템:**
- 좌/우 이동
- 쪼그리기(Duck/Crouch) — 낮은 탄환 회피, 낮은 지형 통과
- 점프 — 장애물 회피, 적 머리 위 통과, 높은 탄환 회피

### 2. 사격 (Shooting)

**핵심 원칙: 이동 중 동시 사격.**

이것이 런앤건을 플랫포머와 구분짓는 가장 중요한 요소.

**조준 방향 선택 (장르 내 변형):**
- **한정 조준 (Metal Slug 방식)**: 8방향. 가장 일반적. 위/아래/대각선 포함, 이동 방향과 독립.
- **자유 조준 (Contra 방식)**: 8방향 이동 중 8방향 사격. 더 많은 전술 유연성.
- **전방 자동 조준**: 일부 캐주얼 런앤건에서 사용. 사격 방향 고정 = 접근성 향상.

**기본 무기 (Default Weapon):**
- 반드시 존재해야 함 — 무제한 탄약.
- 상대적으로 약하지만 언제나 사용 가능.
- 특수 무기 탄약 소진 시 자동 복귀.

### 3. 적 설계 (Enemy Waves & Patterns)

**런앤건의 적 설계 원칙:**
- 밀집된 다수 적 — "파고들고 처리하는" 전투.
- 결정론적 패턴(랜덤 없음) — 학습 가능, 암기 보상.
- 피격 즉시 대응 — 반응 속도 요구.

**적 계층 구조 (대부분의 런앤건에서 공통):**
- **소형 잡몹(Grunts)**: 다수, 저체력, 빠른 처리.
- **중형 적(Elite)**: 더 높은 체력 또는 특수 공격.
- **포탑/시설물(Installation)**: 고정, 피격 주의 필요.
- **보스(Boss)**: 스테이지 끝. 복수 페이즈, 패턴 암기 필수.

### 4. 피격 판정과 결과 (Hit Consequence)

**장르 표준: 1히트 킬 (One-Hit Kill)**
→ [[Arcade Difficulty Design]] 참조

핵심 효과: 모든 순간이 집중을 요구. 실수 비용 최대화.
현대적 변형: 체력바(2~3히트) 또는 무적 구간(i-frame).

**필수 피드백 요소:**
- 피격 시 시각 효과 (캐릭터 깜빡임/폭발)
- 피격 시 음향 효과 (즉각적 오디오 피드백)
- 사망 애니메이션 (명확한 상태 전환)
- 재시작까지의 시간 최소화

### 5. 스테이지 구조 (Stage Structure)

**기본 런앤건 스테이지:**
- 선형 진행 (좌→우). 역행 불가 또는 최소화.
- 중간 장애물/구조물 — 수직 이동 통로.
- 중간 미니보스 (선택).
- **스테이지 끝 보스** — 필수. 이것이 장르의 페이스 리셋 포인트.

**스테이지 수**: 일반적으로 5~8스테이지. Metal Slug: 6스테이지 표준.

## 베이스 시스템 간 상호작용

```
[이동] ←→ [사격] : 이동 중 사격이 핵심
   ↕              ↕
[점프]    ←→   [적 패턴] : 점프로 탄환 회피
                  ↕
            [피격 결과] : 패턴 암기 → 생존
```

이 5요소의 **긴장 관계**가 런앤건의 재미를 만든다:
- 이동을 멈추면 적에게 쉽게 피격.
- 달리면 조준이 어렵고 함정에 빠질 위험.
- 사격에 집중하면 이동 관리 소홀.
- → 세 행동을 동시에 최적화해야 하는 것이 핵심 도전.

## 선택적 기반 요소 (Optional But Common)

이 요소들은 기반은 아니지만 장르에서 매우 보편적:

| 요소 | 대표 게임 | 없을 때 영향 |
|---|---|---|
| 2인 협동 | Contra, Metal Slug, 모든 주요 타이틀 | 단독 플레이로도 성립하나, 재미 감소 |
| 무기 픽업 시스템 | Contra, Metal Slug | 기본 무기만이면 단조로움 빠름 |
| 수류탄/보조 무기 | Metal Slug | 없어도 성립 (Contra는 B-배리어만) |
| 목숨 시스템 | 모든 아케이드 런앤건 | 무제한 리트라이로 대체 가능 (Cuphead 방식) |
| 스코어링 | 아케이드 버전 | 현대 콘솔에서는 선택적 |

## 런앤건 구현 체크리스트

my-game이 런앤건을 구현할 때 최소 필요한 것:

```
필수 (MUST HAVE):
☐ 횡스크롤 2D 이동 (좌/우/쪼그리기)
☐ 점프 (단일 점프 최소; 더블 점프는 선택)
☐ 이동 중 사격 (전방 최소; 8방향 권장)
☐ 기본 무기 (무제한 탄약, 언제나 사용 가능)
☐ 다수 적 웨이브 (패턴 학습 가능한 배치)
☐ 스테이지 구조 + 보스
☐ 명확한 피격 피드백 + 재시작 메커니즘

권장 (SHOULD HAVE):
☐ 특수 무기 픽업 시스템 (3~5종)
☐ 수류탄/보조 공격 (전술 선택지 추가)
☐ 2인 협동 플레이
☐ 다양한 적 유형 (소형/중형/포탑/보스)
☐ 스테이지 내 환경 다양성 (지형, 배경 변화)

선택 (NICE TO HAVE):
☐ 탈것/슬러그 시스템
☐ 스코어링 시스템
☐ 분기 경로
☐ 숨겨진 요소 (Metal Slug POW 등)
```

## Implications For my-game

1. **5가지 기반 시스템부터 먼저 만든다**: 무기 픽업, 탈것, 협동보다 먼저 "이동+점프+사격+적 패턴+피격 결과"가 재미있어야 한다. 기반이 재미없으면 확장으로 구제할 수 없다.
2. **사격하면서 이동하는 것이 자연스러워야 한다**: 컨트롤 폴리싱의 최우선 순위. 이것이 어색하면 장르 아이덴티티가 없다.
3. **보스는 필수**: 스테이지 끝 보스 없이는 진행의 리듬이 무너진다. 보스 1개만 있어도 런앤건이지만 보스 없이는 다른 장르다.

## Open Questions
