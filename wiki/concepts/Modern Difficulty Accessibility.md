# Modern Difficulty Accessibility

## 개념 정의

**현대 난이도 접근성(Modern Difficulty Accessibility)**은 인디 액션 게임이 2018년 이후 수렴하고 있는 난이도 설계 철학이다. 핵심은 **코어 경험의 도전성을 유지하면서도 진입 장벽을 낮추는 옵션을 제공**하는 것이다. 아케이드 원형의 "단일 난이도"와 완전히 다르다.

아케이드 난이도 원형의 현대적 계승에 대해서는 [[Arcade Difficulty Design]] 참조.

---

## 1. 역사적 전환점

### Celeste (2018) — 인디 접근성의 분기점

Celeste의 Assist Mode는 현대 인디 액션 접근성의 레퍼런스 모델이 됐다:

- **무적 토글**: 피격 데미지 없음
- **게임 속도 감소**: 60%~100% 슬라이더
- **스태미나 무제한**: 벽 점프 무한
- **프레이밍 중요성**: "이것은 게임을 망치는 게 아니라, 더 많은 사람이 경험을 즐기게 하는 도구"

이 프레이밍 방식이 이후 수많은 인디 게임에 영향. "Assist Mode"라는 명칭 자체가 관용구가 됨.

### Another Crab's Treasure (2024) — 아이러니한 접근성

"One-shot weapon"(일격 무기) 접근성 옵션: 임의의 적을 즉시 처치. The Game Awards 2024 접근성 부문 주목. 1히트 즉사 '역활용' — 어려움을 느끼는 플레이어에게 '즉사 무기'를 주는 방식.

---

## 2. 2024–2026 접근성 옵션 표준 패턴

인디 액션 타이틀이 점진적으로 채택하는 패턴:

### Tier 1 — 난이도 프리셋 (가장 기본)
- Easy / Normal / Hard (또는 다른 명칭)
- Xbox Accessibility Guideline 108: 이상적으로 4개+ 프리셋 권장 (의무 아님)
- 설명 문구: "이 모드는 탐험을 즐기려는 플레이어를 위함" 등 — 가치 판단 없는 설명

### Tier 2 — 개별 메커닉 조정
- 적 체력 슬라이더
- 적 공격 빈도 슬라이더
- 플레이어 데미지 스케일링 (받는 데미지 50% 등)
- 자동 저장 간격 조정

### Tier 3 — 어시스트 토글
- 무적 / 갓 모드
- 시간 제한 비활성화
- 자동 에임 지원
- 배리어/방어막 제공

### Tier 4 — 구조적 우회
- 스테이지 스킵
- 자동 진행(Auto-play)
- 적 없음 모드 (탐험·스토리 집중)

> [!gap] 2024~2026 인디 런앤건 중 Tier 3~4를 채택한 타이틀의 정확한 비율 데이터 없음.

## 3. 1히트 즉사와 현대 접근성의 관계

### 1히트 즉사가 '수용되는' 조건

| 조건 | 근거 |
|---|---|
| **즉시 재시작** (1초 이내) | Hotline Miami / Katana Zero — 리트라이 마찰 없음이 핵심 |
| **결정론적 패턴** (랜덤 없음) | 죽음이 "학습"이지 "운"이 아니어야 수용 |
| **시각적 텔레그래프** | 적 공격 예고 동작 — "피할 수 있었다" 감각 |
| **최소 1개 난이도 옵션** | Cuphead Simple, Katana Zero 슬로다운 — 불가능한 벽이 아님을 보여줌 |

### 1히트 즉사가 '거부되는' 조건

| 조건 | Steam 포럼 패턴 |
|---|---|
| 즉시 재시작 없음 (로딩 포함) | "죽을 때마다 기다려야 한다" — 좌절 급증 |
| 랜덤 패턴 적 | "억울한 죽음" — 학습 불가 |
| 유일 난이도 + 진입 불가 | "게임이 나를 원하지 않는다" 반응 |
| 3D 공간 + 깊이 판단 오류 | [[Arcade Difficulty Design]] § 3D 실패 케이스 |

### 체크포인트 빈도와 1히트 즉사의 결합

1히트 즉사의 "공정함"은 체크포인트 빈도에 의존한다:

```
Hotline Miami: 방(Room) = 체크포인트 단위. 즉사 → 방 시작부터 재시작 (5~30초).
Katana Zero: 룸 단위 즉시 재시작. 거의 동일 구조.
Cuphead: 보스 시작부터 재시작. 보스 전투 자체가 1~3분 → 허용 범위.
Contra (아케이드): 목숨 소진 = 스테이지 처음. 컨티뉴로 복귀.
```

**핵심 원칙**: 즉사 → 재시작 지점까지 거리가 길수록 좌절 증가. "방 단위" 또는 "체크포인트 30초 이내"가 현대 플레이어 허용 범위.

## 4. 하이브리드 사례: 1히트를 "피한" 성공 인디

| 타이틀 | 체력 시스템 | 특징 |
|---|---|---|
| Cuphead | 1히트 즉사 (기본) + Simple 모드 (체력 부여) | 1히트 코어 유지, 모드로 완충 |
| Hollow Knight | 마스크 시스템 (복수 히트) + Soul으로 회복 | 1히트 아님, 단 데미지 누적 |
| Dead Cells | 체력 + 아이템으로 회복 | 로그라이트 + 체력바 혼합 |
| Hades 2 | God Mode: 피해 저항 20%→80% 점진 증가 | "사망마다 더 쉬워지는" DDA |
| Pizza Tower | 일반 구간: 데미지 없음(점수 감소만). 보스전: 체력바 | 장르 특화 하이브리드 |

## 5. 산업 표준 현황 (2025–2026)

### 강제 요건이 아닌 권고

- **Xbox XAG 108**: 다수 난이도 프리셋 권장. **의무 인증 요건 아님**.
- **Sony PlayStation**: 별도 접근성 의무 기준 없음 (2026 현재).
- **Steam**: 접근성 태그 선택 가능 — 자발적 선택.

### 리뷰 문화의 변화

- 2023~2026 Steam/Metacritic 리뷰: 난이도 옵션 부재를 감점 요소로 언급하는 빈도 증가.
- 그러나 **장르 팬의 기대치는 다름**: 런앤건 팬은 1히트 즉사를 "장르 특성"으로 수용. 접근성 요구는 일반 플레이어층에서 더 강함.
- **IGDA / GDC 2025**: "Accessibility by Design" 접근 — 기능 추가가 아닌 설계 원칙으로 접근성 통합.

> [!gap] "2025~2026 인디 런앤건 중 난이도 옵션 제공 비율" 공식 통계 없음.

## 6. Implications For my-game

### 권고 설계 (1히트 즉사 선택 시)

```
코어 경험: 1히트 즉사 + 즉시 재시작 (1초 이내)
옵션 A: Easy 모드 = 2~3히트 체력바 추가 (Cuphead Simple 모델)
옵션 B: 데미지 스케일링 슬라이더 (받는 데미지 50%)
체크포인트: 방/섹션 단위, 30초 재시작 이내
텔레그래프: 모든 적 공격에 예고 동작 필수
```

### 피해야 할 패턴

- 단일 난이도 + 리뷰어 접근 불가 구간: "너무 어렵다" 리뷰 집중 위험
- 1히트 즉사 + 로딩 재시작: 리트라이 마찰이 장르 정체성을 파괴
- "Baby Mode" 류 명칭: 접근성 옵션을 선택하는 플레이어를 비하하지 말 것

### 최소 기준선 (2026 기준)

Xbox 콘솔 출시를 고려한다면: **2개 이상 난이도 프리셋 + 각 프리셋의 차이 설명** 필수.
Steam Only라면: 1개 난이도로도 기술적으로 가능하나 마케팅 약점.
