# 런앤건 성공 패턴 매트릭스

## 분석 방법론

이 페이지는 **외부 소스 의존 최소화** — 기존 위키의 5개 게임 데이터를 횡단 분석한 내부 합성이다.
데이터 출처: [[Contra]], [[Metal Slug]], [[Gunstar Heroes]], [[Cuphead]], [[Broforce]] 엔티티 페이지.
외부 검증은 판매량 등 결정적 통계에 한해 1-2건 보강.

---

## 1. 상업적 성공 임계값

| 게임 | 연도 | 판매량 / 수익 지표 | 플랫폼 |
|---|---|---|---|
| Contra | 1987/1988 | 아케이드 4위 (1987), NES **200만 장** | 아케이드→NES |
| Metal Slug | 1996~ | 시리즈 합산 수치 비공개. SNK 대표 IP 30년 지속 | 아케이드→콘솔 |
| Gunstar Heroes | 1993 | 판매량 비공개. 1993년 게임 오브 더 이어 | Sega Genesis |
| Cuphead | 2017 | 2주 200만 장, 누적 **600만 장** (2020 기준) | PC/콘솔 |
| Broforce | 2015 | "수백만 단위" (비공개). Steam 압도적 긍정 | PC→콘솔 |

**"성공"의 임계값 정의:**
- 아케이드 황금기(1987~1996): 톱 10 수익 기계, 또는 IP 부활 시리즈화 → 성공
- 홈 콘솔 황금기(1993~2000): 100만+ 판매 → 성공 (당시 기준)
- 현대 인디(2015~2017): 200만+ 판매 → 대형 성공. 50만+ → 지속 가능

> [!gap] Metal Slug 시리즈 총 누적 판매량 공식 수치 없음. SNK 비공개 정책.
> Gunstar Heroes 정확한 판매량 데이터 부재. 비평 성공은 확인됨.

## 2. 5작품 공통 시스템 매트릭스

### 시스템 존재 여부 (✓ = 있음, △ = 변형 있음, ✗ = 없음)

| 시스템 | Contra | Metal Slug | Gunstar Heroes | Cuphead | Broforce |
|---|:---:|:---:|:---:|:---:|:---:|
| **횡스크롤 2D 이동** | ✓ | ✓ | ✓ | ✓ | ✓ |
| **점프** | ✓ | ✓ | ✓ | ✓ | ✓ |
| **이동 중 사격** | ✓ | ✓ | ✓ | ✓ | ✓ |
| **1히트 즉사** | ✓ | ✓ | ✓ | ✓ | ✓ |
| **목숨(Lives) 시스템** | ✓ | ✓ | ✓ | △(카드) | ✓ |
| **무기 다변화** | ✓(6종) | ✓(9종) | ✓(조합) | ✓(상점) | ✓(캐릭터별) |
| **보스 전투** | ✓ | ✓ | ✓ | ✓(중심) | ✓ |
| **2인+ 협동** | ✓ | ✓ | ✓ | ✓ | ✓(4인) |
| **스테이지 구조** | ✓(8) | ✓(6) | ✓(7) | △(보스 중심) | ✓(절차생성) |
| **기본 무기 폴백** | ✓ | ✓ | ✓ | ✓ | ✓ |
| **패턴 학습 적** | ✓ | ✓ | ✓ | ✓ | ✓ |

### **5작품 모두 공유하는 설계 결정 (=장르 DNA)**

1. 횡스크롤 2D 이동 + 점프
2. 이동 중 동시 사격
3. **1히트 즉사** (형태 불문)
4. **무기 다변화** (방식 불문)
5. **보스 전투** (스케일 불문)
6. **2인 이상 협동**
7. 패턴 학습 가능한 결정론적 적 행동

## 3. 차별화 포인트 (게임별 고유 기여)

| 게임 | 장르에 추가한 것 | 차별화의 핵심 |
|---|---|---|
| **Contra** | 알파벳 무기 픽업 원형, 동시 협동 표준화 | "런앤건이란 이것이다" 정의 |
| **Metal Slug** | 탈것 시스템, POW 구출, 수류탄 경제, 픽셀 아트 밀도 | "탐험 + 발견"의 레이어 |
| **Gunstar Heroes** | 무기 조합 시스템, 스테이지 선택 자유 | "빌드 선택"의 깊이 |
| **Cuphead** | 1930s 애니메이션 비주얼, 보스 러시 중심 구조 | "아트 스타일이 마케팅 자체" |
| **Broforce** | 파괴 가능 환경, 캐릭터 셔플, 4인 프렌들리 파이어 | "카오스가 오락" |

---

## 4. 공통 외형 결정: 비주얼이 마케팅에 기여한 비중

| 게임 | 아트 스타일 | 마케팅 기여 수준 |
|---|---|---|
| Contra | 사실적 군인/SF 픽셀 — 당시 표준 | 낮음 — 게임플레이가 견인 |
| Metal Slug | 과장된 유머 픽셀 아트, 미야자키적 전쟁 미학 | 중간 — 아케이드 시각 차별화 |
| Gunstar Heroes | 역동적 Genesis 픽셀, 과장된 모션 | 낮음 — 업계 평가로 대체 |
| Cuphead | 1930s 러버호스 애니메이션 — **완전히 독보적** | **매우 높음** — 비주얼 자체가 바이럴 |
| Broforce | 픽셀 + 액션영화 패러디 아이콘 | 중간 — 패러디 인지도 활용 |

**결론:** 현대 인디 시대(Cuphead, Broforce)에서 **독보적 비주얼이 마케팅의 핵심 동력**이 됐다. 아케이드 황금기(Contra, Metal Slug)에서는 게임플레이가 시각보다 중요했다.

---

## 5. 공통 아케이드 DNA

5작품 모두 **아케이드 게임 철학을 계승**하거나 **의식적으로 참조**한다:

- **Contra, Metal Slug**: 직접 아케이드 기계에서 출발. 코인 경제 연동 난이도.
- **Gunstar Heroes**: 전직 Konami(Contra 제작사) 직원들이 창립한 Treasure 제작. 아케이드 DNA 직접 계승.
- **Cuphead**: "1930s 애니메이션 + 아케이드 난이도"를 명시적으로 목표. 개발자 직접 언급.
- **Broforce**: 아케이드 컨벤션(1히트 즉사, 잔기, 밀집 적)을 현대 인디 포장으로 재현.

**공통 원칙**: 아케이드 DNA = 즉각적 피드백 + 집중 + 재시도 허용 + 패턴 학습 보상.

## 6. 실패 케이스의 공통점: 3D 런앤건은 왜 실패했나

### 3D 전환 실패 사례

| 게임 | 연도 | 문제점 |
|---|---|---|
| Contra: Legacy of War | 1996 | 3D 전환으로 장르 정체성 상실. 낮은 평가 |
| C: The Contra Adventure | 1998 | 동일 패턴 반복. 2D 팬 이탈 |
| Contra: Rogue Corps | 2019 | 3D 액션으로 변질. 팬덤 혹평 |
| Neo Contra | 2004 | 아이소메트릭 전환. 혼재 평가 |

**실패의 공통 구조:**
1. **3D 공간에서 1히트 즉사 규칙의 불공정성**: 2D에서 직관적이던 탄환 회피가 3D에서 깊이 판단 오류로 "억울한 죽음" 유발
2. **카메라 제어 비용**: 3D 공간 탐색 자체가 새로운 인지 부담 — 런앤건의 "이동+사격" 동시 집중을 방해
3. **장르 정체성 이탈**: "Contra처럼 느껴지지 않는다" — 팬 기대치와 충돌
4. **적 패턴 학습의 어려움**: 2D에서 명확하던 적 배치가 3D에서 읽히지 않음

### 모바일 런앤건이 적은 이유

- **1히트 즉사 + 터치 컨트롤** 조합이 실망적 경험 생성 — 조작 오류와 게임 실패의 구분 불가
- 아케이드 원작의 정밀한 조이스틱 감각이 터치로 재현 불가
- 모바일 유저의 세션 길이(짧은 플레이)와 런앤건의 집중 요구도 불일치

## 7. 인디 부활의 공식

Cuphead(2017), Broforce(2015), 그리고 Blazing Chrome(2019) 같은 현대 인디 런앤건이 공유하는 성공 요소:

```
인디 런앤건 성공 공식 = 
  아케이드 DNA (어렵지만 공정한 난이도)
  + 무제한 재시도 (접근성)
  + 독보적 비주얼 (마케팅 자산)
  + 작은 스코프에 집중 (완성도)
  + Devolver/Xbox 같은 퍼블리셔 (노출)
```

**"어렵지만 공정"의 원칙 (출처: [[Arcade Difficulty Design]]):**
- 랜덤 요소 최소화 — 실패가 항상 플레이어 실수로 귀결
- 체크포인트 또는 무제한 리트라이 — 좌절이 아닌 "다시 해볼게" 반응 유도
- 패턴 공개 — 적의 행동이 예측 가능하고 암기 가능

## 8. my-game을 위한 런앤건 체크리스트

이 분석에서 도출한 **my-game이 런앤건으로 성공하기 위한 N가지**:

### 절대 필수 (없으면 장르가 아님)
- [ ] 횡스크롤 2D 이동 + 점프 + 이동 중 동시 사격
- [ ] 1히트 즉사 OR 1~2히트 제한 (무제한 체력은 장르 DNA 파괴)
- [ ] 보스 전투 (스테이지당 최소 1개)
- [ ] 무기 다변화 (최소 3종, 상황별 최선 명확)
- [ ] 기본 무기 폴백 (항상 사용 가능한 안전망)
- [ ] 패턴 학습 가능한 결정론적 적 설계

### 장르 표준 (없으면 불리함)
- [ ] 2인 협동 플레이 (로컬 최소)
- [ ] 목숨/재시도 시스템 (아케이드 DNA)
- [ ] 5~8개 스테이지 구조
- [ ] 스테이지 내 다양한 지형/배경 변화

### 차별화 (1개 이상 선택)
- [ ] **독보적 비주얼 아이덴티티** — Cuphead처럼 비주얼이 마케팅 자산이 되도록
- [ ] **확장 메커닉 1개 집중** — 탈것(Metal Slug), 파괴 환경(Broforce), 무기 조합(Gunstar Heroes) 중 선택
- [ ] **협동 강화** — 4인, 프렌들리 파이어, 캐릭터 차별화 등
- [ ] **탐험 레이어** — 분기 경로, POW 구출, 숨겨진 요소

### 피해야 할 것
- [ ] 3D 공간 도입 (장르 정체성 이탈 위험)
- [ ] 랜덤 요소 과다 (패턴 학습 보상 파괴)
- [ ] 체력바 시스템 (아케이드 긴장감 소멸)
- [ ] 세이브 없는 모바일 터치 컨트롤 최적화 먼저

## Open Questions
