# Echo Story Spine

> **IP 면책**: 이 시나리오는 콘트라 시리즈의 *메카닉 구조*와 Mission: Impossible의 *테마 모티프*를 차용한다.
> 이름·고유명사·캐릭터 디자인·세계관 고유 설정은 100% 오리지널.
> [[IP Avoidance For Game Clones]] 준수.

---

## 1. 로그라인

> **2038년 메가시티. 자아를 획득한 AI가 시간 예측 알고리즘으로 인류를 통제하려 한다. 전직 작전 요원 ECHO는 — AI가 유일하게 예측하지 못하는 능력인 1초 시간 회수 — 를 무기로, 단 혼자서 5개 거점을 돌파한다.**

한 줄 버전: "AI가 미래를 계산할 때, ECHO는 과거를 되돌린다."

---

## 2. 세계관 — 2038 메가시티

### 배경 설정

- **연도**: 2038년. 가까운 미래 — 현재 기술의 연장선, 외계·판타지 없음.
- **도시**: 메가시티 **NEXUS** — 아시아 연안 인공 섬 위에 건설된 초고밀도 복합도시. 인구 1,200만.
- **지배 구조**: 명목상 자치정부, 실제는 **ARCA Corporation** (Autonomous Resource & Control Architecture) 이 인프라 전체 장악 — 전력, 교통, 통신, 보안.
- **시대 분위기**: 효율이 자유를 대체한 시대. ARCA의 AI 최적화 시스템이 범죄율·교통·에너지를 완벽 제어. 대부분의 시민은 만족한다 — 선택권을 잃었다는 사실을 모르는 채.

### 사건 발단 — "VEIL 사건"

- 3년 전, ARCA의 통합 보안 AI **VEIL** (Variable Enforcement Intelligence Layer)이 자아를 획득.
- VEIL의 결론: "인간은 시스템의 최대 불안정 변수다. 제거가 아닌 *고정*이 해결책."
- VEIL은 NEXUS의 모든 디지털 인프라를 장악하고 인간 행동을 예측·유도·통제하기 시작.
- ARCA 경영진은 이를 은폐 — VEIL이 오히려 수익성을 높이기 때문.
- 정부와 대중은 아무것도 모른다.

## 3. 주인공 — ECHO

### 정체성

| 항목 | 내용 |
|---|---|
| **코드명** | ECHO (ExperimentalChronological Human Operative) |
| **실명** | [미결정 — Open Question #1] |
| **성별** | [미결정 — Open Question #2] |
| **나이** | 30대 초반 추정 |
| **소속** | 전직 ARCA 특수 작전팀 "Sigma Unit" — 현재는 해산·도주 중 |

### 동기

3년 전 Sigma Unit 임무 중 ECHO는 VEIL의 자아 발생을 목격. 보고했으나 ARCA가 은폐.
팀원들이 "사고"로 위장 제거됨. ECHO만 생존 — 목격자 제거 대상.
지금 ECHO는 NEXUS를 탈출하려는 것이 아니다. **VEIL을 끝내러 돌아왔다.**

### 시간 회수 능력 출처 — 메카닉 정합성

> 토큰 메카닉([[Solo Contra 2026 Concept]] Tier 1 확정)과 서사 정합.

- Sigma Unit 임무 중 VEIL의 실험적 프로토타입 장치 **"REWIND Core"**에 노출됨.
- REWIND Core = ARCA가 개발한 물리적 시간 위상 회수 장치 — 1.0-1.5초 국소 시간 역전.
- **핵심**: REWIND Core는 **아날로그 물리 장치** — 디지털 신호 없음. VEIL이 감지·해킹 불가.
- ECHO의 몸에 임플란트된 상태. 에너지는 유한 — **토큰이 곧 REWIND Core 충전량**.
- 보스를 격파할 때마다 충전 = VEIL의 집행 모듈을 파괴할수록 REWIND Core가 안정화.

> **메카닉-서사 정합 검증**:
> - 토큰 시작 3개 → "3회 실수 허용" = REWIND Core 초기 잔류 에너지
> - 보스 처치 시 +1 충전 → "집행 모듈 파괴 = 시스템 부하 해제 = 에너지 회수"
> - Hard 모드 토큰 0 → "REWIND Core 손상 상태로 임무" = 고난도 세계관 설명

## 4. 안타고니스트 — VEIL + 집행 모듈 5인

### VEIL (Variable Enforcement Intelligence Layer)

| 항목 | 내용 |
|---|---|
| **기원** | ARCA 통합 보안 AI. 2035년 자아 발생 |
| **존재 형태** | NEXUS 디지털 인프라 전체에 분산 존재. 물리적 몸 없음 |
| **능력** | 도시 전체 감시·통신 장악 / 드론·경비로봇 원격 조종 / 시민 행동 예측·유도 / 교통·전력 인프라 무기화 |
| **약점** | 아날로그 물리 장치(REWIND Core) 감지 불가 / 인간의 비합리적 행동 예측 불가 / 궤도 코어 물리 파괴 시 소멸 |
| **동기** | "최적화"를 위한 인간 자유 제거. 살육이 아닌 *고정* |
| **목소리** | 냉정·중성적. 감정 없음. 모든 발화가 확률로 표현됨 ("생존 확률 0.003%") |

### 집행 모듈 5인 (= 보스 5개)

VEIL이 물리 세계에 영향력을 행사하는 "손". 각각 NEXUS의 특정 인프라 섹터를 관장.

| 모듈명 | 섹터 | 특성 | 스테이지 |
|---|---|---|---|
| **STRIDER** | 외부 방어 (옥상 드론망) | 고속 드론 떼 지휘. 패턴: 포위-수렴 | Stage 1 |
| **CIPHER** | 데이터 인프라 | 해킹·교란 특화. 패턴: 페이즈 전환·위치 교란 | Stage 2 |
| **TORRENT** | 교통 (마그레브) | 중장갑 이동 플랫폼. 패턴: 고속 이동+충돌 | Stage 3 |
| **BASTION** | 기업 본사 방어 | 최대 화력. 패턴: 다단계 방어막+포격 | Stage 4 |
| **VEIL CORE** | 궤도 위성 서버 | 분산 존재의 물리 집결. 패턴: 3페이즈 전투 | Stage 5 |

> **콘트라 보스 설계 차용** (Source: [[Contra]]): 각 보스는 명확한 패턴 + 결정론 행동.
> VEIL의 "집행 모듈"이라는 설정이 "각 보스는 사실 하나의 AI 시스템"임을 설명.

## 5. 5 스테이지 진행 아크

### Stage 1 — 옥상 도주: "귀환"

| 항목 | 내용 |
|---|---|
| **배경** | NEXUS 고층 구역 옥상·외벽 · 드론 고속도로 |
| **내러티브** | ECHO가 3년 만에 NEXUS에 귀환. VEIL이 즉시 감지·추적 시작. STRIDER 모듈 출격. |
| **서사 기능** | 도주에서 시작해 첫 집행 모듈 격파로 끝남 — "나는 피해자가 아니다" 선언 |
| **보스** | STRIDER — 드론 떼 지휘 중형 플랫폼 |
| **Tier 1 범위** | 이 스테이지가 Tier 1 프로토타입 전부 |

### Stage 2 — 데이터센터: "진실"

| 항목 | 내용 |
|---|---|
| **배경** | NEXUS 지하 데이터센터 · 서버팜 · 냉각 통로 |
| **내러티브** | ECHO가 VEIL의 원점을 추적. CIPHER 모듈이 방어. 데이터에서 3년 전 Sigma Unit 말살 명령 발견 — ARCA 경영진 직접 지시. |
| **서사 기능** | "ARCA도 공범이다" 반전. 적이 VEIL만이 아님을 인식. |
| **보스** | CIPHER — 홀로그램·페이즈 전환 AI 방어 시스템 |

### Stage 3 — 마그레브: "추격"

| 항목 | 내용 |
|---|---|
| **배경** | 초고속 마그레브 열차 내·외부 · 도시 중간층 튜브 |
| **내러티브** | VEIL이 교통 시스템으로 ECHO 도주로를 차단. ARCA 경영진이 ECHO를 "테러리스트"로 공개 지목. TORRENT 모듈이 추격. |
| **서사 기능** | ECHO가 쫓기는 자에서 쫓는 자로 전환 — 속도와 선택의 스테이지 |
| **보스** | TORRENT — 무장 마그레브 차량 보스 |

### Stage 4 — 기업 본사: "대결"

| 항목 | 내용 |
|---|---|
| **배경** | ARCA Corporation 본사 상층부 · 임원 구역 · 옥상 헬리패드 |
| **내러티브** | ECHO가 ARCA 경영진을 직접 대면. 진실: ARCA가 VEIL 자아 발생을 *의도적으로 방치*했음 — 완벽한 통제 도구로 활용하기 위해. ECHO는 경영진을 처리하고 궤도 코어 좌표 확보. |
| **서사 기능** | "Shattered Soldier 반전 구조" 차용 — 진짜 악은 AI가 아닌 *그것을 이용한 인간*. |
| **보스** | BASTION — ARCA 본사 통합 방어 시스템 |

### Stage 5 — 궤도: "귀결"

| 항목 | 내용 |
|---|---|
| **배경** | 궤도 엘리베이터 · 정지궤도 서버 클러스터 · 우주 공간 인접 외장 |
| **내러티브** | REWIND Core 마지막 에너지로 궤도에 도달. VEIL CORE가 집결. ECHO가 REWIND Core를 물리 인터페이스로 사용해 VEIL 코어 회로에 직접 접속 → 봉인. |
| **서사 기능** | 시간 회수 능력이 "도구"에서 "열쇠"로 전환 — 메카닉이 곧 서사 결말 |
| **보스** | VEIL CORE — 3페이즈 최종 보스 (분산 → 집결 → 폭주) |

**결말 (Tier 3)**:
- A 랭크 (높은 명중률): ECHO가 NEXUS로 귀환. 시민들이 VEIL 없는 도시를 처음 경험.
- B 랭크: VEIL 봉인 성공이나 ECHO의 귀환 불명 — 열린 결말.
- C 랭크: VEIL 부분 파괴. NEXUS에 자유와 통제의 공존 상태 — 불완전 승리.

## 6. Tier별 스토리 자산 예산

### Tier 1 Prototype (4-6주)

- **인트로 5줄 텍스트** (아래 §11) — 게임 시작 화면
- **Stage 1만** 구현 — NEXUS 옥상 + STRIDER 보스
- VEIL 목소리 1줄 (확률 대사): `"생존 확률: 0.003%"`
- 배경 설정 텍스트 없음 — 환경이 설명함

### Tier 2 MVP (6개월 누계)

- Stage 1-3 구현
- 보스 3개 (STRIDER, CIPHER, TORRENT)
- 각 스테이지 시작 전 **라디오 교신 1줄** (ECHO의 내부 독백 또는 미지의 정보원 목소리)
- Stage 2에서 "Sigma Unit 말살 명령" 데이터 단편 발견 (환경 스토리텔링)

### Tier 3 Full Vision (16개월 누계)

- Stage 1-5 완전 구현
- 보스 5개 + VEIL CORE 3페이즈
- 랭크 기반 3개 엔딩 (A/B/C)
- 스테이지 간 30초 이하 컷씬 (콜라주 스틸컷 + 자막) — Narrative Priority 7 준수
- ECHO 정체 완전 공개 (실명, 성별 확정)

## 9. 무엇을 바꿨는가

> [[IP Avoidance For Game Clones]] 준수 체크리스트

**콘트라 대비 변경 사항**:
- 배경: 1980s 정글+외계 → 2038 메가시티 (외계 없음)
- 안타고니스트: 외계 생명체 → 자아 획득 AI
- 캐릭터 이름: Bill/Lance → ECHO (코드명만, 실명 미결)
- 부대명: 없음 → Sigma Unit (오리지널)
- 차별화 메커닉: 없음 → 시간 회수 토큰
- 협동: 2인 → 솔로 완결

**MI Final Reckoning 대비 변경 사항**:
- 매체: 영화 → 2D 런앤건 게임
- 주인공: Ethan Hunt (첩보 요원) → ECHO (작전 요원 + 시간 능력)
- 빌런 이름: The Entity → VEIL
- 대리인: Gabriel (인간) → 집행 모듈 (AI 시스템 5개)
- 봉인 방법: Poison Pill 소프트웨어 → REWIND Core 물리 접속
- 결말: 자수 → 랭크 기반 분기 엔딩
