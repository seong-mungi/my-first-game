# Indie Self Publishing Run and Gun

## 개념 정의

**인디 런앤건 자가퍼블리싱(Indie Self-Publishing Run and Gun)**은 외부 퍼블리셔 없이 개발팀이 직접 Steam 등 플랫폼에 출시하는 런앤건 타이틀의 상업적 현실을 다룬다. 이 페이지는 "퍼블리셔 없이 얼마나 팔 수 있는가"에 대한 근거 기반 분석이다.

---

## 1. 퍼블리셔 정의: 경계 명확화

### 퍼블리셔 역할 분해

| 역할 | 퍼블리셔 | 마케팅 에이전시 | 플랫폼(Steam/Xbox) |
|---|---|---|---|
| 개발 자금 지원 | ✓ (주요) | ✗ | ✗ |
| QA 지원 | ✓ | ✗ | ✗ |
| 피처링/쇼케이스 | ✓ | △ | ✓ (Steam 에디터스 초이스 등) |
| 마케팅 캠페인 | ✓ | ✓ | △ |
| 콘솔 인증 지원 | ✓ | ✗ | ✓ (ID@Xbox 등) |
| 지역화 | ✓ | △ | ✗ |
| PR / 언론 | ✓ | ✓ | ✗ |

### 자가퍼블리싱 정의 (이 페이지 기준)

**협의**: Steam Direct 출시 + 모든 마케팅을 개발팀 자체 수행. 외부 자금 없음.

**광의**: 소형 유통사(예: Fangamer — 피지컬 유통만)는 허용. 마케팅·자금 지원 제공 파트너 없음.

Pizza Tower의 경우: Fangamer가 피지컬 에디션 담당. 디지털 Steam 출시는 Tour De Pizza 단독 → **광의 자가퍼블리싱**으로 분류.

> [!gap] "소형 퍼블리셔"와 "유통 파트너"의 경계가 실제 계약 내용에 따라 달라 공개 데이터로 판단 어려움.

---

## 2. 상업 임계값 매핑

### 런앤건/인접 장르별 판매 데이터

| 타이틀 | 자가퍼블리싱 여부 | 런앤건 해당 | 판매/수익 | 비고 |
|---|---|---|---|---|
| Cuphead (2017) | ✗ (Microsoft Studios) | ✓ (하이브리드) | 600만 장 (2020) | Xbox 독점 마케팅 |
| Hotline Miami (2012) | ✗ (Devolver Digital) | △ (탑다운) | 5백만 장+ (2022) | Devolver 초기 대표작 |
| Broforce (2015) | ✗ (Devolver Digital) | ✓ | 수백만 (비공개) | Devolver 마케팅 |
| Katana Zero (2019) | ✗ (Devolver Digital) | △ (혼합) | 50만 장 (1년 이내) | Devolver 2위 빠른 Switch 출시 |
| Pizza Tower (2023) | **✓** (Tour De Pizza) | ✗ (플랫포머) | ~1M~2M 소유자 추정 | $25M 수익 추정 |
| Vampire Survivors (2022) | **✓** (poncle) | ✗ (Bullet Heaven) | 6백만 장 추정 | £40M 순자산 추정 (2024) |
| Blazing Chrome (2019) | ✗ (The Arcade Crew 퍼블리싱) | ✓ | 비공개 (소형) | 소형 퍼블리셔 |
| Huntdown (2020) | ✗ (Coffee Stain Publishing) | ✓ | 비공개 | 소형 퍼블리셔 |

### 임계값 분석

```
200만 장 임계값:
  런앤건 자가퍼블리싱 → 현재까지 확인된 사례 없음

100만 장 임계값:
  자가퍼블리싱 + 인접 장르(플랫포머, Bullet Heaven) → Pizza Tower, Vampire Survivors
  순수 런앤건 자가퍼블리싱 → 미확인

50만 장 임계값:
  Katana Zero가 Devolver 퍼블리싱으로 1년 내 달성
  자가퍼블리싱 런앤건에서의 달성은 공개 데이터 없음

10만~20만 장 임계값:
  자가퍼블리싱 런앤건의 현실적 기대치
  소형 퍼블리셔 런앤건(Blazing Chrome, Huntdown)의 추정치
```

## 3. 자가퍼블리싱 성공의 구조 분석

### Pizza Tower 케이스 스터디 (가장 근접 사례)

- **개발**: McPig(디자이너) + Sertif(프로그래머) 주도. **5년 개발**.
- **커뮤니티**: itch.io 무료 데모로 수년간 팬 빌드업 → 출시 전 Reddit/Discord 입소문.
- **마케팅 대체**: $0 마케팅 예산 → 높은 완성도 + 커뮤니티 생성 입소문.
- **Steam 평가**: 출시 1일 내 Steam 최고 평점 게임 진입. 2024년 기준 Steam 최고 평점 2D 플랫포머.
- **수익**: 첫 달 $3M, 연간 $25M 추정.
- **한계**: 런앤건이 아닌 플랫포머. 이동 중 사격 없음.

### Vampire Survivors 케이스 스터디 (자가퍼블리싱 극한 성공)

- **개발**: Luca Galante 1인 개발 (초기), 이후 팀 확장.
- **출시**: Early Access $2.99 → 바이럴. 유튜버/스트리머 자연 확산.
- **퍼블리셔**: poncle = 개발사 = 퍼블리셔. 100% 자가퍼블리싱.
- **수익**: 6백만 장 추정, $20M+ 수익 추정.
- **한계**: 장르가 전혀 다름(Bullet Heaven). 런앤건의 정밀 사격 컨트롤 vs 자동 공격.

## 4. 퍼블리셔 효과: 정량화 시도

### Devolver Digital 효과

Devolver가 퍼블리싱한 런앤건/인접 인디의 공통 기여:

| 기여 항목 | 효과 |
|---|---|
| Steam 출시 타이밍 최적화 | 게임 페스티벌(PAX, GDC) 연동 |
| 트레일러 제작 지원 | 고품질 프로모 자료 |
| 언론 PR | IGN, Kotaku 등 리뷰어 사전 접근 |
| 인디 페스티벌 부스 | PAX West/East, EGX 등 물리적 노출 |
| 콘솔 인증 지원 | Switch, PS4 포팅 가속 |
| 소셜미디어 앰플리피케이션 | Devolver 팔로워 수백만 → 트레일러 확산 |

> [!gap] Devolver가 구체적으로 매출의 몇 %를 기여했는지 공개 데이터 없음. "Devolver 없었으면 Hotline Miami가 얼마나 팔렸는가"는 반사실.

### 마케팅 등가 효과

인플루언서 캠페인, 인디 페스티벌 노출, 콘솔 인증이 퍼블리셔 없이도 가능한 경로:

| 경로 | 비용/조건 | 퍼블리싱 대체 가능성 |
|---|---|---|
| 스트리머/유튜버 자연 확산 | 0비용 but 제어 불가 | 높음 (Pizza Tower 사례) |
| Steam Next Fest 참가 | 무료 (선발 경쟁) | 중간 |
| ID@Xbox 프로그램 | 무료 (심사 필요) | 중간 (콘솔 인증만) |
| 인디 게임 페스티벌 직접 출품 | 출품비 $50~500 | 낮음 (규모 한계) |
| 마케팅 에이전시 고용 | $10k~100k+ | 중간 (비용 부담) |
| itch.io 데모 선공개 | 무료 | 높음 (장기 전략) |

## 5. 현실 체크: 런앤건 퍼블리셔 없는 성공 장벽

### 구조적 어려움

1. **장르 팬의 인식 문제**: 런앤건은 소수 장르. Devolver 같은 큐레이터가 "이 게임을 알아야 할 사람들"에게 도달시키는 역할을 함.
2. **콘솔 출시 비용**: Switch/PS5 인증 = 개발 비용 + 사무 부담. 소형 퍼블리셔가 이 부담을 대신.
3. **리뷰어 접근성**: 무명 자가퍼블리셔는 대형 미디어 리뷰어 접근이 어려움. Devolver는 신뢰 브랜드.
4. **트레일러 클릭**: 유명 퍼블리셔 로고가 있는 트레일러가 더 높은 클릭률.

### 자가퍼블리싱으로 장벽을 넘는 조건

```
조건 1: 독보적 비주얼 — 트레일러 하나로 퍼블리셔 없이도 바이럴
   (Cuphead: 비주얼이 마케팅. 단 Xbox 지원 있었음)
   (Pizza Tower: 5년 커뮤니티 + itch.io 데모)

조건 2: 스트리머/유튜버 플레이어블
   Vampire Survivors: 시청하기 재미있는 게임 → 자연 확산
   런앤건: 관전 재미가 높은 장르 (보스 패턴 공략, 노데미지 런)

조건 3: 가격 접근성
   Vampire Survivors: 초기 $2.99 — 클릭 장벽 최소화
   런앤건: 통상 $15~25 — 더 높은 장벽

조건 4: 커뮤니티 선행 빌드업
   Pizza Tower: itch.io 데모 수년 → 출시 전 팬덤 형성
```

## 6. 모바일 런앤건 자가퍼블리싱

> [!gap] 모바일에서 자가퍼블리싱 런앤건으로 200만 다운로드를 달성한 사례는 공개 데이터에서 확인되지 않음.

- **구조적 한계**: 모바일 런앤건은 [[Run and Gun Success Pattern Matrix]] §6에서 설명한 터치 컨트롤 문제로 장르 자체가 소형 시장.
- **자가퍼블리싱 모바일**: 구글플레이/앱스토어의 알고리즘 피처링 없이 200만 다운로드 달성은 사실상 불가능.

## 7. Implications For my-game

### 현실적 목표 설정

| 시나리오 | 조건 | 예상 판매 |
|---|---|---|
| 자가퍼블리싱 + Steam Only | 독보적 비주얼 + 커뮤니티 빌드업 | 5만~20만 장 |
| 자가퍼블리싱 + Steam + 스트리머 자연 확산 | 플레이어블 + 시청 재미 | 10만~50만 장 |
| 소형 퍼블리셔 (The Arcade Crew, Playism 등) | 마케팅 파트너 + 콘솔 지원 | 5만~20만 장 |
| Devolver급 퍼블리셔 | 선택받아야 함 | 50만~200만 장 |

### 권고 전략

1. **itch.io 데모 선행 공개**: Pizza Tower 모델. 비용 없이 팬덤 빌드업.
2. **비주얼 우선 투자**: 트레일러 하나로 공유되는 비주얼 — 마케팅 예산을 대체.
3. **소형 퍼블리셔 탐색**: Devolver 불필요. 콘솔 인증 + 언론 PR만 담당하는 파트너.
4. **스트리머 플레이어블 설계**: 보스 패턴 공략 / 노데미지 런 / 속도 도전 — 관전 가치 높은 콘텐츠.
5. **200만 장 목표는 단계적으로**: 첫 작품에서 10만 장 → IP 인지도 → 후속작 확장.
