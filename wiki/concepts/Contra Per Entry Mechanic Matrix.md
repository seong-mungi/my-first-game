# Contra Per-Entry Mechanic Matrix

Cross-entry comparison of the **one or two systems each Contra entry layered onto the 1987 baseline** (run / jump / 8-way shoot / weapon-letter swap / 1-hit death). This is a reuse menu, not a history.

|------|-------|-------------|------------------------|-------------|---------|
| 1987 | Contra (NES/Arcade) | 2D side + pseudo-3D | Baseline + pseudo-3D base stages | Pseudo-3D dropped | Canon |
| 1988 | Super Contra | 2D side + top-down | Top-down stage variants | Top-down dropped post-Contra III | OK |

## Survival Rule

> [!key-insight] What Contra teaches about innovation
> 1. **Stay 2D side-scroll**. Every 3D / non-side perspective entry (1996, 1998, 2019) underperformed and was followed by a 2D return.
> 2. **Add one mechanical axis at a time**. Branching (Hard Corps), grading (Shattered Soldier), grappling (Contra 4), perks (Galuga) — each was a single new axis. Rogue Corps stacked four (perspective + classes + cooldown + twin-stick) and broke.
> 3. **Survival requires reuse**. A system that returns in two later entries (sliding, dual-weapon, character select) is validated. Single-entry experiments (Mode 7, Hit Rate) are interesting but optional.
