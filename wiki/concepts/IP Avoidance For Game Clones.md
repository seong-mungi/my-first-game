# IP Avoidance For Game Clones

## Definition

게임 클론을 합법적으로 제작·상업 출시하기 위한 IP 회피 전략. 기본 원칙은 단순하다: **"게임 메카닉은 저작권 보호 대상이 아니므로 자유 모방 가능, 표현(캐릭터·로고·음악·트레이드드레스)은 모두 교체해야 한다"**. 미국과 한국의 저작권법이 공유하는 표현/아이디어 이분법(idea-expression dichotomy)에서 직접 도출되는 결론이다 (Source: [[Metal Slug IP Avoidance Guide]]).

## 두 축의 분리

| 축 | 보호 여부 | 모방 가능성 |
|---|---|---|
| **메카닉(아이디어)** | 보호 없음 | 자유 |
| **표현(비주얼·이름·음악)** | 보호됨 | 금지 또는 교체 |

이 분리는 직관적이지만 실무에서 자주 흐려진다. 흐려지는 지점이 곧 법적 위험 지점이다.

## 보호되지 않는 것 (자유 모방)

다음은 모두 메카닉/아이디어 영역으로, 어떤 게임에서든 그대로 가져와도 법적 문제가 없다:

- 게임 규칙·시스템 (이동·점프·사격·픽업·차량·HP·잔기 등)
- 장르 컨벤션 (런앤건의 횡스크롤, 보스 멀티페이즈, 1히트 즉사 등)
- 일반적 캐릭터 설정 (군인·악당 장군·외계 침략자)
- UI/HUD 레이아웃의 구조 (탄약 표시·잔기 카운터·HP바)
- 음악 장르 스타일 (군악·록 퓨전 — 멜로디 자체가 아니라 *장르 스타일*만)
- 스테이지 구조 (사막·정글·도시 등 환경 컨벤션)

## 교체 필수 (표현)

다음은 표현 영역으로, 모방 시 침해 위험이 발생한다:

| 카테고리 | 예시 |
|---|---|
| 캐릭터 디자인·이름 | Marco Rossi(메탈슬러그), Mario(닌텐도) |
| 시그니처 차량/오브젝트 실루엣 | SV-001 탱크의 둥근 포탑·짧은 차체 |
| 로고·게임 타이틀 | "Metal Slug", "Contra" 등 상표 등록된 명칭 |
| 음악·효과음 | OST 멜로디·시그니처 사운드 |
| 적 캐릭터 구체 디자인 | 미라·외계 침략자의 *특정 외형* |

판단 기준은 **식별 가능성(recognizability)**: 평균 플레이어가 "이거 X 게임이네"라고 알아본다면 침해 위험이 있다.

## 트레이드드레스 함정

저작권 외에도 **트레이드드레스(trade dress)** — 시각적 정체성의 종합적 인상 — 가 있다. 개별 요소는 모두 교체했더라도 *전체적 인상*이 원작과 거의 같다면 침해 가능성이 남는다.

대표적 트레이드드레스 요소:

- 게임의 색채 팔레트 시그니처
- 대표 캐릭터의 실루엣 비례
- UI 폰트·아이콘 스타일의 종합
- 박스아트·프로모션 비주얼의 구도

> [!gap] 트레이드드레스 침해는 사례별 판단이며 정량 기준이 없다. 글로벌 출시 시 변호사 검토가 가장 중요한 단계.

## 안전 등급 매트릭스

| 등급 | 전략 | 위험 |
|---|---|---|
| 🟢 **오리지널 클론** | 메카닉 모방 + 비주얼·이름 100% 오리지널 + "inspired by" 명시 | 매우 낮음. [[Broforce]] 모델 |
| 🟡 **장르 컨벤션 활용** | 메카닉 모방 + 장르 컨벤션 캐릭터 + 약간 다른 비주얼 | 낮음. 평균 인디 출시 |
| 🟠 **시각적 헌사(homage)** | 위와 같으나 원작을 명백히 연상시키는 디테일 잔존 | 중. 변호사 검토 필수 |
| 🔴 **팬아트 클론** | 원작 캐릭터·이름 그대로 사용 | 매우 높음. 비상업만 묵인 |
| ⛔ **무단 자산 사용** | 원작 스프라이트·음악·코드 직접 사용 | 즉시 침해 |

## 실증 사례

- 🟢 [[Broforce]] (Free Lives, 2015) — Contra·Rambo·액션영화 패러디 기반의 오리지널 클론. Steam 상업 출시 성공.
- 🔴 GitHub의 다수 메탈슬러그 클론 — "metal slug" 이름 유지, 비상업 묵인 영역에 머무름.
- 🟢 *Vampire Survivors* (poncle, 2022) — *Magic Survival* 메카닉 클론 + 자체 비주얼·캐릭터 → 글로벌 히트.
- 🟢 *Stardew Valley* (ConcernedApe, 2016) — *Harvest Moon* 메카닉 클론 + 오리지널 표현 → Harvest Moon 권리사와 무분쟁.

## 절차 (제작 시 체크리스트)

1. **메카닉 단계.** 원하는 만큼 모방. 메카닉 모방 자체는 평판 리스크일 수는 있어도 법적 리스크는 없다.
2. **표현 단계.** 모든 비주얼 자산을 처음부터 자체 제작 또는 라이선스 자산으로 충당.
3. **이름 단계.** 원작 IP 이름(타이틀·캐릭터·장소) 사용 금지. 마케팅 카피에서도 주의.
4. **음악 단계.** 오리지널 작곡 또는 CC0/Royalty-free.
5. **트레이드드레스 점검.** 박스아트·프로모션 자료·UI 종합 인상이 원작과 충분히 다른지 외부 시점으로 검토.
6. **출시 전 변호사 검토.** 상업 출시 전 IP 전문 변호사 1회 검토 비용은 분쟁 비용 대비 매우 저렴.

## Failure Modes

- **"이건 패러디니까 괜찮아" 가정.** 미국 패러디 항변(parody defense)은 좁다. 한국에서는 더 좁다. 패러디로 묵인되는 범위를 과대평가하지 말 것.
- **"비상업이니까 괜찮아" 가정.** 묵인은 권리 포기가 아니다. 권리자가 언제든 통지(takedown)할 수 있다.
- **자산 출처 추적 부재.** 외주·AI·인터넷에서 가져온 자산에 원작 IP 일부가 섞여 있어도 책임은 출시사에게 있다.
- **메카닉 특허.** 드물지만 일부 게임 메카닉은 *특허*로 보호된다(예: 닌텐도의 일부 시스템). 저작권과 별개로 특허 검색이 필요한 경우가 있다.

## Implications For my-game

본 프로젝트가 런앤건(메탈슬러그형)을 채택할 경우:

- 8대 핵심 시스템(이동·무기·근접·차량·HP·적·스코어·레벨)은 자유 구현 (참고: [[Metal Slug IP Avoidance Guide]]).
- 메탈슬러그 비주얼·이름·음악은 100% 교체.
- 모범 모델은 [[Broforce]] — 패러디 + 오리지널 게임플레이.
- v1 GDD에 IP 회피 체크리스트 섹션을 명시적으로 임베드.
- 상업 출시 결정 시 IP 전문 변호사 1회 검토 단계를 마일스톤으로 추가.

장르 결정이 라인디펜스([[Plants vs Zombies]] 모방)나 Battle Cats형([[Battle Cats]] 모방)이라도 동일 원칙이 적용된다.
