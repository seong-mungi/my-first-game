# Wikipedia — 코나미 코드 (소스 페이지)

**URL:** https://en.wikipedia.org/wiki/Konami_Code
**접근일:** 2026-05-08

## 추출 핵심 정보

### 기원
- **창안자:** 하시모토 카즈히사 (Kazuhisa Hashimoto), 코나미 프로그래머
- **목적:** Gradius NES 포팅 테스트 시 "게임이 너무 어려워 플레이 불가" 문제 해결
- **유지 이유:** 대량생산 준비 단계에서 발견. 제거 시 새 버그 도입 위험 → 유지 결정

### 게임 외부 이스터에그 목록 (Wikipedia 확인)
- Netflix, Discord, Twitter/X
- Google Assistant, Amazon Alexa, Siri
- ESPN, Facebook
- Bank of Canada 공식 웹사이트
- 영화 Wreck-It Ralph (2012) 직접 등장

### 북미 대중화 경로
1. Gradius NES (1986) — 기술적 최초 등장
2. **Contra NES (1988) — 30 lives 코드로 대중화 (별칭: "Contra Code")**
3. 코나미 자사 게임 전반으로 확산
4. 타사 게임 이스터에그로 독립 확산
5. 현대 디지털 플랫폼으로 진출
