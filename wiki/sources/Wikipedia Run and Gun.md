# Wikipedia: Run and Gun Video Game

## Source Type
Encyclopedia reference — stable, broadly accurate for historical facts and genre taxonomy.

## Key Claims

- **Definition**: Run and gun is a sub-genre of shoot 'em ups where the protagonist travels on foot. Emphasis is on simultaneous movement and shooting — "no camping."
- **Genre mix**: Combines platformer (jumping, navigation) with shoot 'em up (shooting waves of enemies). Notable for frequent cooperative gameplay.
- **Subtypes**: Side-scrolling (most common), vertically scrolling overhead (Commando, Ikari Warriors), isometric.
- **First side-scroller**: Jump Bug (1981).
- **Genre-defining moment**: Contra (1987) — combined platforming with 8-directional shooting; set the template.
- **Hardware lineage**: Arcades (1980s) → home consoles (NES/SNES/Genesis/Neo Geo) → handheld → indie revival (Steam/Switch, 2010s–present).
- **Genre evolution**: Stealthily evolved into third-person shooter (Cabal → Wild Guns → modern TPS).
- **Failed 3D transition**: Genre "never really successfully passed the Video Game 3D Leap" — remained a 2D genre.
- **Key titles by era**: Gun Fight (1975), Commando (1985), Ikari Warriors (1986), Contra (1987), Gunstar Heroes (1993), Metal Slug (1996), Cuphead (2017), Blazing Chrome (2019).
