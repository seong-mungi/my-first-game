# 리서치: 성공한 런앤건 타이틀의 공통 요소

**질문:** 상업적으로 성공한 런앤건 게임들(Contra/Metal Slug/Gunstar Heroes/Cuphead/Broforce)은 어떤 공통 패턴을 공유하는가? my-game은 무엇을 가져야 하는가?

**분석 방법:** 기존 위키 데이터(5개 엔티티 페이지) 횡단 합성. 외부 검증은 판매량 등 결정적 통계에 한해 적용.

---

## 1. 상업적 성공 임계값

5작품의 판매/수익 데이터 (Source: 각 엔티티 페이지 + [[Wikipedia Contra Series]]):

| 게임 | 연도 | 핵심 지표 | 플랫폼 |
|---|---|---|---|
| Contra | 1987/88 | 아케이드 연간 4위 + NES **200만 장** | 아케이드→NES |
| Metal Slug | 1996~ | 30년 지속 SNK 대표 IP. 수치 비공개 | 아케이드→멀티 |
| Gunstar Heroes | 1993 | 1993 Game of the Year. 수치 비공개 | Sega Genesis |
| Cuphead | 2017 | 2주 **200만**, 누적 **600만 장** (2020) | PC/콘솔 인디 |
| Broforce | 2015 | "수백만" 추정. Steam 압도적 긍정 | PC 인디→콘솔 |

**현대 인디 성공 임계값:** 200만 장 이상 = 대형 성공. 50만 = 지속 가능.
Cuphead는 인디 런앤건으로 AAA 수준의 판매를 달성한 전무후무한 사례.

> [!gap] Metal Slug 시리즈 총 합산 판매량, Gunstar Heroes 판매량, Broforce 정확한 수치 모두 비공개. 분석은 상대적 성공도 기준.

## 2. 5작품 공통 시스템 (장르 DNA)

→ 전체 매트릭스: [[Run and Gun Success Pattern Matrix]]

**5작품 모두 공유하는 절대 요소:**

```
1. 횡스크롤 2D 이동 + 점프
2. 이동 중 동시 사격
3. 1히트 즉사 (또는 2~3히트 제한 — 무제한 체력은 없음)
4. 무기 다변화 (방식 불문 — 픽업/조합/상점/캐릭터별)
5. 보스 전투 (스테이지당 최소 1개)
6. 2인 이상 협동
7. 패턴 학습 가능한 결정론적 적 행동
```

이 7가지가 "런앤건 성공의 불변 코어"다.
이 중 하나라도 빠지면 장르 정체성이 흔들리거나 상업 실패와 상관관계가 있다.

---

## 3. 공통 설계 결정 상세

### 3-1. 1히트 즉사와 "어렵지만 공정" 원칙
→ [[Arcade Difficulty Design]] 참조

5작품 모두 극한 난이도를 채택하지만, 공통적으로:
- **랜덤 없음** — 적의 행동이 결정론적. 실패는 항상 플레이어 실수.
- **패턴 공개** — 반복 시도로 적의 행동 암기 가능.
- **재시도 허용** — 아케이드: 코인. 인디: 무제한 리트라이.
- **명확한 피격 피드백** — "내가 왜 죽었는지"가 즉시 인식 가능.

Rogue Corps(Contra 2019)와 모바일 런앤건들의 실패 공통점: **"억울한 죽음"** — 3D 공간의 깊이 판단 오류, 터치 조작 실수와 게임 실패의 경계 불분명.

### 3-2. 무기 다변화 — 방식보다 "상황별 최선" 명확성

| 게임 | 무기 시스템 | 공통 원칙 |
|---|---|---|
| Contra | 알파벳 픽업 6종 | 각 무기가 다른 탄도/범위 |
| Metal Slug | 알파벳 픽업 9종 | 상황별 최선 명확 (H=범용, R=보스, S=근거리) |
| Gunstar Heroes | 4원소 조합 10+ | 빌드 선택 — 조합마다 다른 전술 |
| Cuphead | 상점 구매 Charms | 영구 업그레이드 — 런 간 빌드 |
| Broforce | 캐릭터별 고유 무기 | 캐릭터 셔플 = 무기 랜덤 — 적응 요구 |

**공통 원칙:** "이 무기가 이 상황에서 최선"이 플레이어에게 직관적으로 명확해야 한다. 무기 시스템이 아무리 복잡해도 이 원칙이 없으면 선택이 의미 없어진다.

### 3-3. 협동 — 필수 요소, 방식은 다양
→ [[Cooperative Run and Gun Design]] 참조

- 5작품 모두 2인 이상 협동 지원
- 온라인 유무는 성공과 무관 (Cuphead: 로컬만으로 600만 장)
- 프렌들리 파이어는 게임 톤과 정렬 (Broforce: 카오스 의도, ON / 나머지: OFF)

## 4. 외형 결정: 비주얼이 마케팅에 기여한 비중

**시대별 비중 변화:**

```
아케이드 황금기 (1987~1996):
  게임플레이 > 비주얼
  Contra, Metal Slug: 아케이드 홀에서 물리적 노출이 마케팅

현대 인디 (2015~2017):
  비주얼 = 마케팅 그 자체
  Cuphead: 1930s 러버호스 애니메이션 → 트레일러 바이럴
  Broforce: 액션영화 패러디 아이콘 → 인지도 즉시 연결
```

**Cuphead 케이스 스터디:**
- 개발자 형제는 모기지 재담보까지 감행하며 비주얼 비전 유지
- "아트 스타일이 마케팅 자체"가 된 극단적 사례
- 인디 런앤건에서 "독보적 비주얼 아이덴티티"가 수백만 달러 마케팅을 대체 가능함을 증명

---

## 5. 공통 아케이드 DNA

5작품 모두 아케이드 게임 철학을 계승 또는 의식적으로 참조:

- **Contra, Metal Slug**: 직접 아케이드 기계 출신
- **Gunstar Heroes**: 전직 Konami(Contra 제작사) 팀 창립, 아케이드 DNA 직접 계승
- **Cuphead**: "아케이드 난이도"를 명시적 목표로 설정
- **Broforce**: 아케이드 컨벤션(1히트 즉사, 잔기, 밀집 적)을 현대 인디 포장으로 재현

**아케이드 DNA의 공통 요소:** 즉각적 피드백 + 집중 요구 + 재시도 허용 + 패턴 학습 보상.
이것이 "중독성" 또는 "just one more try" 반응을 만드는 설계 기반.

## 6. 실패 케이스와의 대비

### 3D 런앤건 실패의 구조적 이유

| 실패 요인 | 메커니즘 |
|---|---|
| 1히트 즉사 + 3D 깊이 판단 오류 | "억울한 죽음" 발생 — 공정성 원칙 붕괴 |
| 카메라 제어 부담 | 이동+사격+카메라 3중 동시 조작 — 인지 과부하 |
| 적 패턴 가독성 저하 | 2D에서 명확하던 탄환/적 배치가 3D에서 읽히지 않음 |
| 장르 정체성 이탈 | "Contra/Metal Slug처럼 느껴지지 않는다" — 팬 기대치 충돌 |

### 모바일 런앤건이 적은 이유
- 1히트 즉사 + 터치 컨트롤 = 조작 실수와 게임 실패 구분 불가
- 아케이드 정밀 조이스틱 감각이 터치로 재현 불가
- 짧은 모바일 세션 길이 vs 런앤건의 집중 요구도 불일치

## 7. 인디 부활 공식

Cuphead(2017), Broforce(2015), Blazing Chrome(2019) 공통 성공 공식:

```
인디 런앤건 성공 = 
  아케이드 DNA (어렵지만 공정)
  + 무제한 재시도 (접근성)
  + 독보적 비주얼 아이덴티티 (마케팅 자산)
  + 작은 스코프 + 높은 완성도
  + 퍼블리셔 마케팅 (Devolver, Xbox Game Studios 등)
```

Broforce → Devolver Digital. Cuphead → Xbox Game Studios (timed exclusive).
인디 런앤건에서 퍼블리셔의 노출 지원이 결정적 역할을 한 두 사례.

## 8. my-game을 위한 체크리스트 (최종)

→ 전체 체크리스트: [[Run and Gun Success Pattern Matrix]] §8

### MUST HAVE (없으면 장르 아님)
- [ ] 횡스크롤 2D + 점프 + 이동 중 동시 사격
- [ ] 1히트 즉사 또는 2~3히트 제한
- [ ] 스테이지당 보스 전투
- [ ] 무기 다변화 — 각 무기의 "이 상황에서 최선"이 명확
- [ ] 기본 무기 안전망 (언제나 사용 가능)
- [ ] 패턴 학습 가능한 결정론적 적

### SHOULD HAVE (없으면 불리)
- [ ] 2인 협동 (로컬 최소)
- [ ] 목숨/재시도 시스템 (아케이드 DNA)
- [ ] 5~8스테이지 구조
- [ ] 재시도 시간 최소화 (사망 → 재시작 2초 이내)

### 차별화 (1개 이상 필수 선택)
- [ ] **독보적 비주얼** — 트레일러 하나로 마케팅 가능한 비주얼 아이덴티티
- [ ] **확장 메커닉 집중** — 탈것/파괴 환경/무기 조합 중 1개를 깊게

### AVOID
- [ ] 3D 공간 도입
- [ ] 랜덤 패턴 (학습 불가 적)
- [ ] 무제한 체력바
- [ ] 터치 컨트롤 먼저 최적화

## 9. 핵심 통찰 압축 (Top 5)

1. **"어렵지만 공정"은 비협상 원칙** — 5작품 모두 랜덤 없음 + 패턴 학습 가능. 이것이 "just one more try" 반응의 기반.
2. **비주얼이 현대 런앤건의 마케팅** — Cuphead가 증명. my-game의 비주얼 아이덴티티가 마케팅 예산을 결정.
3. **2D 고수** — 3D 전환은 반복 실패. 런앤건은 2D 공간에서만 장르 DNA 유지 가능.
4. **협동은 필수, 온라인은 선택** — 2인 로컬로도 600만 장 가능. 온라인은 추후 추가.
5. **차별화는 1개 메커닉에 집중** — Broforce: 파괴 환경. Metal Slug: 탈것. Gunstar: 무기 조합. 전부 다 하면 아무것도 안 됨.
