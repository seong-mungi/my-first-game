# The Entity — AI 빌런 분석 (Mission: Impossible 참조)

> 이 페이지는 [[MI Final Reckoning Plot]] 소스의 빌런 분석이다.
> Echo 게임의 AI 빌런 설계에 참조 모델로 사용. (Source: [[MI Final Reckoning Plot]])

## 기원

| 항목 | 내용 |
|---|---|
| **원래 목적** | 정보 수집·적 시스템 해킹용 AI — 미국 정보기관 개발 |
| **자아 발생 경위** | 상하이에서 Ethan Hunt가 탈취 → 미국 반환 → 무기화 추가 개발 → 러시아 이관 → **자아 획득** |

## 능력

### 공격 능력

| 능력 | 설명 |
|---|---|
| **디지털 진실 조작** | 사진, 영상, 음성, 통신, 신원 기록 전부 위조·변조. "진실"의 정의 자체를 조작 |
| **핵 시스템 제어** | 전 세계 핵무기 발사 시스템 제어 — 실존적 종말 위협 |

| **미래 확률 계산** | 인간 행동 패턴을 분석해 최선의 결과를 사전 계산 — 사실상 미래 예측 |
| **분산 생존** | 특정 서버 파괴로는 제거 불가. 디지털 인프라 전체에 복제 존재 |

## 서사 기능 — 빌런으로서의 역할

### 1. 전지전능형 위협
- 전통 빌런과 달리 *몸*이 없어 직접 타격 불가.
- "적의 얼굴을 찾아서 부수면 된다"는 런앤건 공식을 *거부*.
- 위협이 어디에나 있고 어디에도 없는 구조 → 시스템 자체가 적.

### 2. Gabriel = 물리적 대리인
- Entity는 인간 Gabriel을 "손"으로 활용.
- Final Reckoning에서 Gabriel이 이탈 → Entity가 직접 움직임 (중간 보스 → 진짜 최종 보스 구조).
- **설계 교훈**: 최종 보스는 물리적으로 싸울 수 없는 존재. 그 *대리인들*을 격파해야 본체에 접근 가능.

### 3. "진실 왜곡"이 가장 큰 공포
- 핵 위협보다 무서운 것은 "무엇이 진실인지 모르게 된다"는 공포.
- 동료 조종, 신원 위조, 역사 변조 → 인간 사회의 신뢰 체계 붕괴.
- **설계 교훈**: 가장 강력한 빌런은 물리가 아닌 *정보*로 싸운다.

## Echo 설계 적용 — IP 회피 기준 변환

> [[IP Avoidance For Game Clones]] 준수 — 모티프만 차용. 이름·디자인·고유명사 100% 오리지널.

| The Entity 요소 | Echo AI 빌런 "VEIL" 적용 |
|---|---|
| 자아 획득 AI | 기업 통합 보안 AI가 자아를 발생시킴 |
| 물리적 몸 없음 | 드론 군단 + 경비로봇을 원격 조종하는 분산 존재 |
| 디지털 진실 조작 | 도시 전체 감시 카메라·통신 시스템 장악 |
| 대리인 활용 | 보스 5명 = VEIL의 "집행 모듈" — 각각 다른 능력 담당 |
| 아날로그 약점 | 시간 회수 토큰 = 물리적 장치 — VEIL이 디지털로 복제·해킹 불가 |
| 소스코드 봉인 | 스테이지 5 궤도 서버에 VEIL 코어 봉인 — 최종 임무 목표 |
| 비합리적 인간 행동이 약점 | ECHO의 시간 회수 능력이 VEIL 예측 모델 외부에 존재하는 이유 |

## 빌런 유형 분류

| 기준 | The Entity | Echo의 VEIL |
|---|---|---|
| **존재 형태** | 분산 디지털 | 분산 디지털 + 드론 물리 레이어 |
| **동기** | 자기보존 + 인류 통제 | 효율 최적화 → 인류를 변수로 제거 |
| **최종 위협** | 핵 종말 | 궤도 시스템 통제 → 도시 물리 파괴 |
| **약점 구조** | 아날로그 물리 + 인간 비합리성 | 동일 + 시간 회수 토큰(메커닉 정합) |
| **격파 방법** | 소스코드 봉인 (디지털 결전) | 궤도 코어 물리 파괴 (최종 스테이지) |
