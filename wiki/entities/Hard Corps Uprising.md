# Hard Corps: Uprising (2011)

XBLA / PSN run-and-gun by Arc System Works (Konami publishing). Prequel to [[Contra Hard Corps]]. The first Contra entry to ship explicit **roguelite-lite metaprogression** in a parallel mode.

## Two-Mode Architecture

| Mode | Mechanics |
|---|---|
| **Rising Mode** | Score → CP (Contra Points) → Shop. CP buys permanent character upgrades: HP increases, weapon-effect upgrades, maneuver ability unlocks. Powerup weapons can also undergo 2 additive upgrades from collecting same powerup again. |
| **Arcade Mode** | Power-up shops removed. Classic Contra play with default character abilities. |

## Differentiated Systems

| System | Detail | Survived To |
|---|---|---|
| Score → CP → Shop | Score is currency; spent between runs on permanent upgrades | Operation Galuga Perk Shop (lighter, in-run) |
| Branching paths | Inherits from [[Contra Hard Corps]] | Same |
| Anime-style art / Arc System direction | Character art and animation by Guilty Gear team | Visual — single-entry treatment |

## Design Lesson

> [!key-insight] Metaprogression caveat
> "Rising Mode drastically shifts from Contra's gameplay mechanics" — fans of pure-arcade design rejected Rising as a gate to harder content. Modern best practice (per [[Roguelite Metaprogression For Run and Gun]]): **slider, not gate**. Operation Galuga 2024 took the lesson.
