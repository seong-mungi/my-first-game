# Katana Zero (2019)

## Identity

- **Developer**: Askiisoft (인디 스튜디오, Justin Stander 주도)
- **Publisher**: Devolver Digital (자가퍼블리싱 아님)
- **Release**: April 18, 2019 (PC/Mac/Switch); 2021 (Xbox One/PS4); 2024 (iOS via Netflix Games)
- **Genre**: 횡스크롤 액션 플랫포머 / 네오누아르 (런앤건과 유사하나 사격보다 근접전 중심)
- **Influenced by**: Hotline Miami (Devolver, 동일 퍼블리셔)

## 이 페이지가 필요한 이유

Katana Zero는 **1히트 즉사 + 즉시 재시작** 디자인의 현대적 성공 사례다. 접근성 표준화 시대인 2019년에 이 구조로 50만 장을 1년 이내에 팔았다. [[Modern Difficulty Accessibility]] 분석에서 핵심 데이터 포인트.

## 핵심 게임 시스템

### 1히트 즉사 + 즉시 재시작

- **1히트 즉사**: 어떤 피격도 즉사. 체력바 없음.
- **즉시 재시작**: 사망 즉시 방(Room) 처음부터 재시작. 로딩 없음.
- **내러티브 통합**: 사망 = 주인공 Zero의 "정신적 사전 계획" — 리플레이가 스토리 안에서 설명됨. "처벌"이 아니라 "정신 속 시뮬레이션".
- **시간 조작**: 일시정지 + 느린 동작으로 공격 반응 가능 — 1히트 즉사의 난이도 완화 수단.

### 접근성 수단 (1히트 즉사 유지 + 완화)

| 수단 | 효과 |
|---|---|
| **슬로다운 토글** | 전투 중 느린 동작으로 반응 시간 확보 |
| **즉시 재시작** | 사망 패널티 = 시간 낭비 최소화 |
| **체크포인트**: 방 단위 | 패배해도 이전 방부터 재시작 없음 |
| 2024 모바일: 터치 최적화 | Netflix Games 포팅 — 모바일 접근성 긍정 리뷰 |

이 구조가 "1히트 즉사지만 현대 플레이어가 수용하는" 모범이다.

### 결정론적 패턴 설계

개발자 Justin Stander: "플레이어가 자신의 실수를 인식하고 책임지도록 설계." 모든 적 공격이 예측 가능 — 죽음은 학습 가능한 정보.

## 상업 성과

- **1년 이내 50만 장** 판매, **$5M 수익** (Source: Wikipedia Katana Zero)
- **Devolver's 2위 빠른 Switch 출시작** (1위: Enter the Gungeon)
- **Switch 출시 일주일 내 10만 장** 이상
- 2019년 다수 언론의 "Best Indie Games of 2019" 선정
- Metacritic: ~89~91 수준 (복수 플랫폼)

## 비평 수용성

- 호평 키워드: "fluid," "brilliant," "bleak and beautiful"
- 비교 대상: Hotline Miami (Devolver 동일 퍼블리셔) — 긍정적 비교
- 비판: 일부 리뷰어가 제한된 능력 + 퍼즐 구조가 "제한적으로 느껴진다"고 언급
- 1히트 즉사에 대한 리뷰어 부정 언급: 거의 없음 — 즉시 재시작이 이를 상쇄

## Devolver Digital의 역할

Katana Zero는 **Devolver 퍼블리싱 없이는 이 규모에 도달하기 어려웠을 것이다**:
- Switch 판매가 PC 판매를 앞섰다는 보고 — Devolver의 콘솔 인증 지원 핵심
- 2024 Netflix Games 모바일 포팅도 파트너십의 결과
- 자가퍼블리싱이었다면 PC 전용, 소형 판매에 그쳤을 가능성 높음

> [!gap] "Devolver 없이 자가퍼블리싱했을 때의 판매량" 반사실 데이터 없음.

## 런앤건과의 비교

| 요소 | Katana Zero | 런앤건 (Contra/Metal Slug) |
|---|---|---|
| 이동 | 횡스크롤 2D | 횡스크롤 2D |
| 주 공격 | 근접(칼) + 투척 | 원거리(총) |
| 1히트 즉사 | ✓ | ✓ |
| 즉시 재시작 | ✓ (방 단위) | ✓ (Hotline Miami 모델) |
| 무기 다변화 | 제한적 | 핵심 메커닉 |
| 보스 전투 | ✓ | ✓ |
| 패턴 학습 | ✓ | ✓ |

Katana Zero는 런앤건의 친척 — **이동+전투+1히트+즉시재시작 구조를 공유**하지만 총기 중심 아님.

## Implications For my-game

1. **1히트 즉사 + 즉시 재시작 = 현대 수용 가능**: 2019년에도, 2024년 모바일 포팅에서도 수용됨. 단, 즉시 재시작이 필수.
2. **슬로다운 = 런앤건 응용 가능**: "시간 조작"이라는 내러티브 없이도 "Easy 모드에서 적 속도 75%" 등으로 응용.
3. **내러티브로 리트라이를 정당화**: "게임 오버"가 스토리 안에 설명될 때 반복이 덜 좌절스러움.
4. **50만 장은 Devolver 퍼블리싱 결과**: 자가퍼블리싱에서 동일 목표는 현실적이지 않음.
