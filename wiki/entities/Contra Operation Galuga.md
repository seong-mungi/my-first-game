# Contra: Operation Galuga (2024)

## Identity

- **Developer**: WayForward Technologies
- **Publisher**: Konami
- **Release**: 2024년 3월 12일 (PC/Switch/PS4/PS5/Xbox One/Xbox Series X|S)
- **Genre**: 런앤건 — 원작 Contra (1987) 리메이크
- **원작 감독**: WayForward (Contra 4, 2007도 개발한 팀)

## 장르에서의 위치

Operation Galuga는 **원작 Contra 37년 후 현대 하드웨어용 공식 리메이크**다. WayForward는 이미 Contra 4(2007, DS)로 시리즈 부활에 기여한 팀이다.

아케이드/NES 원작의 스테이지 구조와 무기 시스템을 기반으로 하면서, 현대 런앤건의 편의성을 추가했다.

## 핵심 게임 시스템 변화 (원작 대비)

### 원작 계승 요소
- 8방향 이동 + 사격 기본 구조
- 알파벳 무기 픽업 시스템 (M/F/L/S/R/B 계열 유지)
- 1히트 즉사 또는 소수 히트 방식 (설정 선택)
- 2인 협동 플레이
- 원작 스테이지 테마 오마주 (정글, 기지 등)

### 현대화 추가 요소
- **캐릭터 선택**: Bill Rizer / Lance Bean 외 추가 캐릭터 (Ariana, Stanley, Ms. Troopers 등) — 고유 능력 보유
- **스킬 시스템**: 무기 픽업 시 추가 스킬 선택 옵션 — "어떤 능력과 조합할지" 빌드 선택 요소
- **난이도 옵션**: 복수 난이도 — 아케이드 원작 경험부터 캐주얼까지
- **시각 현대화**: 3D 폴리곤 렌더링 + 클래식 런앤건 사이드스크롤 유지

## WayForward의 Contra 시리즈 기여

WayForward는 두 번의 Contra 개발로 시리즈의 "부활 파트너" 역할:

1. **Contra 4 (2007, DS)**: 원작의 2D 사이드스크롤 정신 충실히 계승. 비평 호평. DS 듀얼 스크린 활용.
2. **Contra: Operation Galuga (2024)**: 원작 리메이크. 현대 멀티플랫폼. 스킬/캐릭터 선택 추가.

Contra 시리즈에서 코나미 내부 개발이 아닌 외부 스튜디오(WayForward)가 "정통 후속"을 맡은 희귀 사례 — 게임 IP 라이선스 활용의 현대적 패턴.

## 서열 내 위치

원작 Contra(1987) → 시리즈 진화 → 3D 실패(1996~2004) → Contra 4 부활(2007) → Hard Corps: Uprising(2011) → **Rogue Corps 혹평(2019)** → **Operation Galuga 리메이크(2024)**

Rogue Corps(2019)의 실패 이후 코나미가 선택한 전략: **3D 실험 포기, 원작 2D 리메이크로 회귀**.

## Implications For my-game

1. **리메이크가 IP 재활성화에 유효**: 코나미는 Rogue Corps 실패 후 리메이크로 방향 전환. "최신 기술 + 원작 설계"가 현대 시장에서 통하는 공식.
2. **캐릭터 + 스킬 선택 추가**: 단일 캐릭터 런앤건에 "선택의 깊이"를 더하는 현대적 패턴. my-game 캐릭터 다양성 설계 시 참조.
3. **난이도 다층화**: 원작 충실도를 유지하면서도 접근성 옵션을 추가 — 아케이드 팬과 신규 플레이어 동시 공략.
