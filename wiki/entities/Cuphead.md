# Cuphead (2017)

## Identity

- **Developer**: Studio MDHR (Chad & Jared Moldenhauer, 캐나다)
- **Publisher**: Studio MDHR / Microsoft Studios (Xbox/PC timed exclusive)
- **Release**: September 29, 2017 (PC/Xbox One); 2019 (macOS); 2020 (PS4); 2021 (Switch)
- **Genre**: Run and gun + Boss Rush 하이브리드
- **DLC**: The Delicious Last Course (2022)

## 장르에서의 위치

Cuphead는 2017년 인디 런앤건 부활의 **가장 상징적인 사례**다. 아케이드 황금기 장르를 현대 PC/콘솔 시장에서 대규모 상업적 성공으로 복원했다.

단, Cuphead는 **순수 런앤건이 아닌 Boss Rush가 중심**인 하이브리드:
- 런앤건 런 스테이지: 소수(코인 수집, 짧은 보너스 레벨 성격)
- **보스 러시**: 게임의 80%를 차지. 복잡한 페이즈-전환 보스 전투.

## 핵심 게임 시스템

### 기본 런앤건 메커닉
- 이동(8방향), 점프, 사격(전방 기본), 대시(무적 프레임 포함).
- 무기(Charms): 게임 내 상점에서 코인으로 구매. 여러 슈팅 타입.
- 패링(Parry): 분홍색 표시 오브젝트를 점프+공격으로 튕겨내기 → 슈퍼미터 충전.

### 슈퍼미터 (Super Meter)
- 5장의 카드 심볼로 표시.
- 적 공격 또는 패링으로 충전.
- 가득 차면 강화 공격(Super) 발동.

### 난이도 설계
- **1히트 즉사** — 아케이드 전통의 순수 계승.
- 보스마다 복수의 페이즈 — 패턴 암기 필수.
- Simple / Regular / Expert 세 난이도 제공.
- 정규 엔딩에 Expert 달성 불필요 — 접근성 배려.

### 협동 플레이
- **2인 로컬 협동** (Cuphead + Mugman).
- 온라인 협동 없음 → 일부 비판.

## 디자인 철학

### 1930년대 러버호스 애니메이션
- **Fleischer Studios** (Popeye, Betty Boop)를 "아트 스타일의 자북(magnetic north)"으로 설정.
- 전통 셀 애니메이션 기법 — "의도적 인간적 불완전함" 유지.
- 수채화 배경, 오리지널 재즈 녹음.
- 개발자 형제는 모기지 재담보까지 감행하며 이 비전을 유지.

### 참조 게임
개발자가 직접 명시한 영향: Gunstar Heroes, Contra III, Contra Hard Corps, Super Mario World, Street Fighter III, Mega Man, Thunderforce series.

## 장르에 미친 영향

- 1930s 스타일 + 극한 난이도 조합이 "아트 게임이 꼭 쉬울 필요 없다"는 선례.
- 이후 수많은 인디 게임이 러버호스 미학 채용.
- Boss Rush를 런앤건 장르의 주력 서브구조로 재정립.
- 인디 개발팀에게 "혹독한 난이도 + 정교한 아트"가 상업적으로 성립함을 증명.

## Implications For my-game

1. **Boss Rush as Core Loop**: Cuphead는 "런앤건 런 스테이지 + 보스 러시"를 역전했다. 런앤건 구조에서 보스 전투를 핵심으로 두고 런 스테이지를 보조로 쓰는 것도 유효한 선택.
2. **단일 독창적 아트 비전**: 완성도 높은 하나의 아트 스타일에 집중하면 소규모 팀도 AAA와 경쟁 가능. 1930s 애니메이션이 마케팅 자체를 했다.
3. **극한 난이도 + 공정한 설계**: Cuphead의 난이도는 아케이드식이지만 패턴은 공정하게 학습 가능. "어렵지만 플레이어를 속이지 않는다" 원칙.
4. **온라인 없이도 성공**: 로컬 2인 협동으로도 충분. 온라인 멀티는 필수가 아님.
